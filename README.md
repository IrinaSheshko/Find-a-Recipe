# 🥗 Recipe Finder App

Интерактивное приложение для поиска кулинарных рецептов по всему миру. Проект использует внешнее API для получения актуальных данных о блюдах, калориях и ингредиентах.

## 🚀 [Live Demo](https://irinasheshko.github.io/Find-a-Recipe/)

---

### 🌟 Основные функции:
* **Поиск в реальном времени:** Интеграция с Edamam API для получения тысяч рецептов.
* **Детальная информация:** Отображение калорий, состава ингредиентов и качественных изображений блюд.
* **Избранное (Favorites):** Реализована логика сохранения понравившихся рецептов в `localStorage` (данные сохраняются даже после перезагрузки страницы).
* **Динамичный интерфейс:** Использование видео-фона и современных эффектов для погружения пользователя.

---

### 🛠 Стек технологий:
* **React** (Hooks: `useState`, `useEffect`)
* **Asynchronous API Calls:** fetch, async/await
* **Persistence:** LocalStorage API
* **CSS3:** Видео-фоны, Flexbox/Grid
